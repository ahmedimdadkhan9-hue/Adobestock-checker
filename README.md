# 🎯 Adobe Stock Checker - LM Studio Edition

**100% Offline | 100% Free | Local AI Powered**

একটি powerful offline tool যা Adobe Stock এর জন্য ইমেজ analyze করে।

## ✨ Features

- 🖼️ Image Analysis (Accept/Reject probability)
- 🤖 LM Studio Integration (Local AI)
- 📊 Quality Scoring (0-100)
- 🎨 Uniqueness Detection
- 📋 Rejection Reasons (4টি main reasons)
- 🔧 Improvement Tips
- 💚 100% Free
- 🔒 100% Offline

## 🚀 Quick Start

### Requirements
- LM Studio: https://lmstudio.ai
- Node.js: https://nodejs.org
- RAM: 4GB+
- Disk: 10GB+

### Installation

```bash
git clone https://github.com/YOUR-USERNAME/adobe-stock-checker-lm.git
cd adobe-stock-checker-lm
npm install
npm start
```

Open: http://localhost:3000
